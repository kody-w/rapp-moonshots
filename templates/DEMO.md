# Demonstration

## Hook

What should the audience understand in the first ten seconds?

## Before

Show the limitation or problem.

## Live sequence

1. Start from a clean state.
2. Run the artifact.
3. Show measurable evidence.
4. Demonstrate a failure or adversarial challenge.
5. Recover or explain the boundary.

## Result

State the measured outcome without overclaiming.
