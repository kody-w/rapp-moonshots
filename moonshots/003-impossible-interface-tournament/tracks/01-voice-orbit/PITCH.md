# Pitch: choices you can orbit

Voice-only interfaces make people memorize syntax. Gaze-only interfaces confuse
attention with consent. Voice Orbit uses each modality for what it does well:

1. voice describes the mission,
2. prediction turns the next possibilities into a spatial orbit,
3. coarse webcam attention highlights one large target, and
4. an explicit `select` or deliberate nod commits.

The center is not another button. It is a permanent safe-rest zone. Looking
home relaxes the interface; sensor loss freezes it; stop, cancel, and undo cut
the line.

The result is a complete hands-free route—not a mockup—with no backend, no
dependencies, no raw-media storage, and a deterministic proof mission anyone
can replay. It also tells the truth: webcam aim is coarse, motion fallback is
not eye tracking, and browser speech may be network-backed.

Voice Orbit should win if predictive petals reduce repair and completion time
without producing a single gaze commit. Speed only counts after safety.
