      Privacy, sensing, and recovery boundary

      Ephemeral frames: a 48×36 analysis canvas is cleared every processing turn. Raw pixel bytes are zeroed. Only a derived grayscale comparison survives until the next frame, then is zeroed.

      Honest estimate: FaceDetector supplies a transient face box/head-position proxy when available. Otherwise local frame motion is used. Neither path is precise gaze or identity inference.

      Freshness: new media frames, valid changing image content, and processed estimates expire independently. Any stale signal clears the arm and freezes sensor-derived confirmation.

      Speech boundary: Web Speech may use browser/vendor processing. No transcript, audio, frame, analytics identifier, or application network request is stored or exported.
