Why this is a moonshot

Software portability is usually asserted through packaging. FleetGate turns it into an executable evidence product: attest the source, run the same behavioral contract across operating systems, reject drift, and package the result for independent review.

Read the source Open public receipt Run it yourself

The honest boundary

FleetGate proves tested behavior and reproducibility for reviewed fixtures. It does not prove arbitrary agents are safe, nor does it claim hardware-backed remote attestation.

Source, tests, evidence schema, rollback, and public-safe receipt are included in this repository.
