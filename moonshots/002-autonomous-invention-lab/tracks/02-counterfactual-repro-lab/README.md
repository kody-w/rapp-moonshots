# Track 02 — Counterfactual Repro Lab

## Application

A cross-platform debugger that changes one bounded environmental variable at a
time, reruns a failure, and reports the first repeatable pass/fail flip as causal
evidence.

## Moonshot pitch

CI tells you where software failed. Counterfactual Repro Lab discovers the
smallest environmental difference that caused it.
