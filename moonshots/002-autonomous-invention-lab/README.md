# 002 — Autonomous Invention Laboratory

> Give the fleet six hours to discover three genuinely novel, buildable
> inventions using existing work. Require experiments, prototypes, adversarial
> review, measurable evidence, and a Shark Tank-style final demonstration.

## Objective

Produce three parallel working prototypes, then select and present the strongest
invention using evidence rather than enthusiasm.

## Selection process

Eight independent strategy agents will propose candidates from different
perspectives:

1. developer productivity,
2. distributed agents,
3. personal knowledge,
4. creative media,
5. security and recovery,
6. product/business value,
7. strange combinations of existing projects, and
8. adversarial feasibility.

The top three concepts become independent tracks under `tracks/`.

## Gates

Each finalist must provide:

- a working prototype,
- a falsifiable value hypothesis,
- one meaningful experiment,
- a measurable result,
- an adversarial review,
- rollback instructions, and
- a three-minute Shark Tank demonstration.

## Fleet model

Tracks execute independently. Shared infrastructure is read-only until two
tracks prove they need the same capability.
