# Finalist selection

Eight independent invention strategies were scored against:

- novelty,
- urgent user pain,
- six-hour buildability,
- measurable proof,
- demo magic, and
- compounding value for RAPP.

## Finalists

| Track | Application | Core invention | Why it advanced |
|---:|---|---|---|
| 01 | Resurrection Proof | A recovery drill that proves an agent estate can actually be restored | Binary evidence, urgent pain, unforgettable demo |
| 02 | Counterfactual Repro Lab | A debugger that changes one environmental variable until a failure flips | Converts environment guessing into causal experiments |
| 03 | Ghost Ops | A playable incident simulator where machines and responder agents become characters | Exceptional demo magic with reusable recovery playbooks |

## Alternates

Migration War Room, Second Wind, RoomTone, WishMirror, Threadline, Agent
Passport, and Bounty-to-Cell remain in the backlog.

## Shared application requirement

Each finalist must be usable as an application, not merely a report or command:

- obvious first interaction,
- visible state and progress,
- one end-to-end experience,
- local sample data,
- exportable result,
- measurable experiment,
- adversarial review, and
- a concise explanation of why it is moonshot-scale.
